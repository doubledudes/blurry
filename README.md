# blurry
The browser extension to BLUR unwanted contents on any webpage with the customised blur level.

## Development
1. Run `npm install`
2. Run `npm run build:dev`

## Shipping
1. Bump the version in `dist/manifest.json`
2. Run `npm run build:prod`
